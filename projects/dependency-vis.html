Premise

This project encompasses the idea of my thesis: to help software developers visualize their dependency graphs and details about each and every dependency to quickly determine how good a dependency is. This addresses the problem of developers not taking the time to investigate and research the libraries they depend on, resulting in potential security and legal problems in the future.

Development

  • Made with JavaScript and Node.js
  • Libraries used: axios.js, d3.js, React.js, express.js.
  • Accesses the GitHub API and npm API
