Premise

I wanted to recreate a very simple game in my first language to see how fast and cleanly I can do it. Tic-tac-toe is well-known to be simple and was a prime choice for challenging myself. The initial version of this project was completed within 2 hours once my Java environment was set up. There are plans to improve this game with different varieties of tic-tac-toe such as having multiple tic-tac-toe boards in a game as well as including a tic-tac-toe AI.

Development

  • Made with Java and the Java Swing Library
  • This was a personal challenge to complete within 2 hours.
